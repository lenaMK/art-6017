### Écrire ou adapter un script 

serveur local:

- HTML et CSS: interprétation par le serveur
- JS: requiert exécution du code par une machine (=/= interprétation par un navigateur)

bonnes pratiques

- ne pas essayer d'ajouter des gros blocs
- confirmer le fonctionnement d'un ajout à la fois
- tester à petite échelle, adapter progressivement
- console.log
- commenter son code

→ cours art des données : p5.js pour des interactions centrées sur l’esthétique / environnement de programmation pour la création artistique
